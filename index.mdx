# Welcome to Simili Bot

Simili Bot (v0.1.0) is an intelligent GitHub issue management system powered by AI and semantic search. It helps maintain high-quality issue tracking by automatically detecting duplicates, routing issues to correct repositories, assessing issue quality, and intelligently triaging problems.

## What is Simili Bot?

Simili Bot is a comprehensive solution for managing issues across GitHub repositories using:

- **Semantic Search**: Find related issues using AI-powered embeddings
- **Duplicate Detection**: Automatically identify duplicate issues with confidence scores
- **Intelligent Routing**: Transfer issues to the correct repository based on content and rules
- **Quality Assessment**: Evaluate issue descriptions and provide improvement suggestions
- **Auto Triage**: Suggest appropriate labels based on issue content
- **Multi-Repository Support**: Manage issues across your entire organization
