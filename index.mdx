# Welcome to Simili Bot

Simili Bot (v0.1.0) is an intelligent GitHub issue management system powered by AI and semantic search. It helps maintain high-quality issue tracking by automatically detecting duplicates, routing issues to correct repositories, and intelligently triaging problems.

## Tech Stack

- **Language**: Go 1.24
- **AI Engine**: Google Gemini (embeddings + LLM)
- **Vector Database**: Qdrant
- **GitHub Integration**: GitHub API v3 + GraphQL
- **Deployment**: Docker + GitHub Actions
