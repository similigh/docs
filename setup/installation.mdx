# Installation

Choose the installation method that best fits your needs.

## Prerequisites
1. **GitHub Account**
2. **Google Gemini API Key** ([Google AI Studio](https://aistudio.google.com/app/apikeys))
3. **Qdrant Vector Database** ([Qdrant Cloud](https://cloud.qdrant.io))

---

## Installation Methods

````carousel
### GitHub Action (Easiest)
Runs automatically on issue events.

Create `.github/workflows/simili-bot.yml`:
```yaml
name: Simili Bot
on:
  issues:
    types: [opened, edited]
jobs:
  simili:
    runs-on: ubuntu-latest
    permissions:
      issues: write
      contents: read
    steps:
      - uses: similigh/simili-bot@v0.1.0
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          QDRANT_URL: ${{ secrets.QDRANT_URL }}
          QDRANT_API_KEY: ${{ secrets.QDRANT_API_KEY }}
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
```

<!-- slide -->
### Docker
Containerized for self-hosting.

```bash
docker run -e QDRANT_URL="..." \
           -e QDRANT_API_KEY="..." \
           -e GEMINI_API_KEY="..." \
           -e GITHUB_TOKEN="..." \
           -v $(pwd)/config:/config \
           simili-bot:latest \
           process --config /config/simili.yaml
```

<!-- slide -->
### Binary
Single executable - build from source.

```bash
go build -o simili ./cmd/simili
./simili --version
```
````

## Troubleshooting

<AccordionGroup>
  <Accordion title="API key is invalid">
    Verify your Gemini API key in Google AI Studio and ensure the Generative AI API is enabled in your project.
  </Accordion>
  <Accordion title="Cannot connect to Qdrant">
    Verify your Qdrant URL format (include port 6333) and check that your API key is correct.
  </Accordion>
</AccordionGroup>
