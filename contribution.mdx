# Contribute to Simili Bot

We love contributions! Whether you're fixing bugs, adding features, or improving documentation, your help is welcome.

## Ways to contribute

### 1. Report bugs
Open an issue on GitHub if you find something that isn't working as expected. Please include steps to reproduce the issue.

### 2. Suggest features
Have an idea for a new feature? We'd love to hear it! Open a feature request issue.

### 3. Submit pull requests
If you want to contribute code:
1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a pull request

### 4. Improve documentation
Documentation is key! If you find a typo or think a section could be clearer, feel free to submit a PR for the docs repository.

## Associate Contribution Guidelines

Please read our [Full Contribution Guide](https://github.com/similigh/simili-bot/blob/main/CONTRIBUTING.md) for more details on our development process and coding standards.
