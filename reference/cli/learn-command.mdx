# Learn Command

Index repository documentation to enable hybrid routing.

## Syntax

```bash
simili learn [OPTIONS]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--org` | string | Organization name | Required |
| `--repo` | string | Repository name | Required |
| `--file` | string | File path (README.md, CONTRIBUTING.md, docs/*.md) | Required |
| `--config` | file | Configuration file | `simili.yaml` |
| `--dry-run` | bool | Preview without storing | false |

## Examples

### Index README

```bash
simili learn --org similigh --repo simili-bot --file README.md
```

### Index multiple files

```bash
simili learn --org similigh --repo simili-bot --file README.md
simili learn --org similigh --repo simili-bot --file CONTRIBUTING.md
simili learn --org similigh --repo simili-bot --file docs/architecture.md
```

### Dry-run

```bash
simili learn --org similigh --repo simili-bot --file README.md --dry-run
```

## Configuration

Enable in `simili.yaml`:

```yaml
transfer:
  enabled: true
  llm_routing_enabled: true
  repo_collection: "simili_repos"
```
